# Movie Library (Личная кинотека)

**Автор:** Акулинин Тимур  
**Дата:** 2026-04-28

## Описание

Приложение «Movie Library» — это графический инструмент для ведения личной коллекции фильмов.  
Оно позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга, просматривать их в таблице, фильтровать по жанру и году, а также сохранять данные в файл формата JSON.

Функциональность:
- Добавление фильма с валидацией (год — целое число, рейтинг — от 0 до 10, поля не пусты).
- Отображение всех фильмов в сортируемой таблице.
- Фильтрация по жанру (выпадающий список уникальных жанров) и году.
- Сохранение и загрузка коллекции в/из JSON-файла.
- Защита от некорректного ввода с выводом информативных сообщений.

## Требования к окружению

- Python 3.7 или выше
- Стандартная библиотека Tkinter (обычно включена в установку Python)
